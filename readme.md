# Crusade CRUD App

## What is it

A lazy logging tool so that we can keep track of our Warhammer 40k Crusade Games.

## How to use it

  * Copy it down
  * Pop open a python terminal
  * Run 
  ```python
from app import db
db.create_all()
```
